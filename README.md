## DESCRIPTION

Interactive Quiz App is a dynamic, web-based quiz application designed to test users’ programming knowledge through multiple-choice questions. It fetches real-time questions from the Open Trivia Database (OpenTDB) API, providing a variety of challenging and engaging questions in the programming category.

Features
1. Dynamic Questions
The app retrieves 10 random programming questions from the Open Trivia DB API.
Questions and options are displayed dynamically, allowing for a fresh experience each time the quiz is taken.

2.Multiple-Choice Options
Each question presents four options (one correct and three incorrect).
Options are randomized to ensure unpredictability and fairness.

3.Immediate Feedback
After selecting an answer, users immediately receive feedback:
Correct Answer is highlighted in green.
Wrong Answer is highlighted in red, while the correct answer is also shown.

4.Navigation
A Next button appears after each answer, allowing users to proceed to the next question.
Once all questions are answered, the app displays the user’s final score.

5.Scoring System
Users earn one point for each correct answer.
The final score is displayed as Your Score: X / 10 at the end of the quiz.

6.Responsive Design
The interface is designed to work seamlessly on desktop and mobile devices.
The quiz container is centered on the screen with modern styling and smooth animations.

7.Modern UI/UX
Clean and visually appealing layout using gradient backgrounds, buttons with hover effects, and subtle animations.
Feedback messages and score are displayed prominently to enhance the user experience.

8.Lightweight and Easy to Use
The app uses vanilla HTML, CSS, and JavaScript, making it fast and easy to deploy.
No external frameworks or libraries are required.

Technical Implementation

HTML: Provides the structure of the quiz, including the question area, option buttons, feedback, next button, and score display.

CSS: Implements a modern card-style layout, responsive design, hover effects, and animation for a smooth user experience.

JavaScript:
Fetches quiz data from OpenTDB API.
Randomizes the answer options.
Handles user interactions, including selecting answers, providing feedback, updating scores, and navigating between questions.

Usage

1.Open index.html in any modern web browser.
2.Read the question displayed at the top.
3.Click on the answer you believe is correct.
4.View the feedback immediately and proceed to the next question using the Next button.
5.At the end, view your total score to assess your performance.
